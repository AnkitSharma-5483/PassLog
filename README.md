PassLOG is a simple web-based password manager that helps users securely store and manage login credentials for different websites. The application provides features like saving passwords, editing entries, deleting credentials, and copying information to the clipboard for quick access.

The project is built using React (frontend) and Node.js (backend) with a clean UI and interactive notifications for a smooth user experience.

Features

Save website credentials

Edit stored passwords

Delete saved entries

Copy credentials to clipboard

Toggle password visibility

Toast notifications for actions

Tech Stack

React (Vite)

Node.js

JavaScript

Tailwind CSS

React Toastify
